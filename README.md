# Генератор случайных цитат
Брюханова Софья, ЭФБО-02-24, Вариант 3. 
Веб-приложение для генерации случайных цитат с возможностью получения новой цитаты по нажатию кнопки.

## Основные возможности

- **Случайные цитаты** - Показ случайной цитаты при каждой загрузке
- **Обновление по клику** - Получение новой цитаты по нажатию кнопки
- **Адаптивный дизайн** - Корректное отображение на всех устройствах
- **Анимации** - Плавные переходы и hover-эффекты

## Стек технологий

- **Frontend**: React 18.2.0
- **Стилизация**: CSS3
- **Язык**: JavaScript (ES6+)
- **Управление состоянием**: React Hooks (useState)
- **Сборка**: Create React App

## Установка и запуск

### Предварительные требования
